#THINGSdb

中国语言编程器

主要组件:

    1,THINGSdb      负责数据存取;
    2,SDP解释器      负责sdp解析执行;
    3,Stanford NLP  负责自然语言解析（他社）;
    4,J2Cache       二级缓存（他社）;

【THINGSdb】

    是一个key-value数据库
    主要记忆数据与数据关系
    被设计为大数据库

【SDP解释器】

    自定义查询程序=SDP
    将自然语言转成SDP
    执行解释SDP

【Stanford NLP】

    感谢斯坦福大学提供的组件。
    将一句中文进行词法+语法分析。
    针对输出的语法树，进行入库处理
    
【J2Cache】

    感谢J2Cache的组件提供。
    由于THINGSdb的速度太慢，不得不追加缓存机制。
